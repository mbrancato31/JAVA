program with the purpose of testing the vigenere enryption.
to run use the command line, direct to the where the program and the test files are,
k4.txt = key, p4.txt = message.
then in the command line write:
- javac vigenere.java (this is to compile the program, press enter)
- java vigenere k4.txt p4.txt (this will run the program using the test key and message file and output the result)
